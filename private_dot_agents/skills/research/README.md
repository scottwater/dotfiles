# Research Skills

Specialized skills for codebase research, pattern discovery, and web research. Adapted from the Katalog workflow agents.

## Skills

**Usage:** `/skill research:<skill-name>`

- `codebase-analyzer` - Analyzes implementation details, traces data flow, explains how code works
- `codebase-locator` - Locates files, directories, and components relevant to a feature or task
- `codebase-pattern-finder` - Finds similar implementations, usage examples, and existing patterns
- `web-search-researcher` - Researches web sources for documentation, best practices, and solutions
- `workflow-analyzer` - Deep dives on research documents to extract high-value insights
- `workflow-locator` - Discovers relevant documents in workflow/ directories
